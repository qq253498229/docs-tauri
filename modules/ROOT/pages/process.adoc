[[process]]
= process

对当前进程执行操作。

当 `tauri.conf.json` 中的 `xref:configuration.adoc#build-config[build.withGlobalTauri]` 设置为 `true` 时，也可以通过 `pass:[window.__TAURI__.process]` 访问此包。

[[functions]]
== Functions

[[exit]]
=== exit

- exit(``exitCode?``: `number`): `{promise_url}:[Promise]`<``void``>

立即以给定的 `exitCode` 退出。

示例

[source,typescript]
----
import { exit } from '@tauri-apps/api/process';
await exit(1);
----

Since: 1.0.0

参数

[%autowidth]
|===
|名称|类型|默认值|描述
|`exitCode`|`number`|`0`|要使用的退出代码。
|===

*返回*: `{promise_url}:[Promise]`<``void``>

一个表示操作成功或失败的 Promise。
