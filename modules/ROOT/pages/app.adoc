= app

link:https://tauri.app/v1/api/js/app[原文地址]

获取应用元数据。

当 `tauri.conf.json` 中的 `xref:configuration.adoc#build-config[build.withGlobalTauri]` 设置为 true 时，该包也可以通过 `pass:[window.__TAURI__.app]` 访问。

必须将 API 添加到 `tauri.conf.json` 中的 `xref:configuration.adoc#allowlist-config[tauri.allowlist.app]`：

[source,json]
----
{
  "tauri": {
    "allowlist": {
      "app": {
        "all": true, // enable all app APIs
        "show": true,
        "hide": true
      }
    }
  }
}
----

建议仅将您使用的 API 加入允许列表，以实现最佳的包大小和安全性。

[[get-name]]
=== getName

 getName(): Promise<string>

获取应用名称

示例:

[source,typescript]
----
import { getName } from '@tauri-apps/api/app';
const appName = await getName();
----

Since: 1.0.0
